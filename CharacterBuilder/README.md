# Character Builder

A standalone 3D character builder tool for positioning character parts and defining bullet spawn points.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Usage

- **WASD** - Move camera
- **Q/E** - Move camera up/down
- **Right-click + Drag** - Rotate camera
- **Shift** - Speed boost
- **Left-click** - Select part
- **Left-click + Drag** - Move selected part
- **Right-click** - Deselect

## Features

- Edit character body parts
- Add custom boxes, spheres, and cylinders
- Place bullet spawn node with direction arrow
- Export character configuration as JSON
